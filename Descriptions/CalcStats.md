# Description

Your task is to process a sequence of integer numbers to determine the following statistics:

* minimum value
* maximum value
* sum of values
* average value

# Sample
```
Values: 1, -1, 2, -2, 6, 9, 15, -2, 92, 11
Input: Min Output: -2
Input: Max Output: 92
Input: Sum Output: 131
Input: Average Output: 13.1
```
